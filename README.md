# Project_InternetOfThings
This project is about Internet of Things.
Ues this project to learn Internet of Things.

Used language: C++ java mysql Vue3 and so on.

vue3 Front-end project + SpringBoot And Maven Back-end project + MySQL database + Linux + Docker Compose to run.
Get Data by stm32 and send to Back-end project.

make sure you have installed docker and docker compose.
make sure that ip of projects are in the right network-configuration.
